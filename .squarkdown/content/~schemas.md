# Squarkup Schemas

Stranger Quarkdown provides a [JSON Schema<sup>↗</sup>](json-schema.org) to validate your `squarkup.json` against. This also adds helpful hover tooltips in IDEs.

Link a schema to your `squarkup.json`:

```json
// squarkup.json
{
  "$schema": "https://sup2point0.github.io/stranger-quarkdown/squarkup-schema/latest.json"

  // ...
}
```

Below you can access past versions of the Squarkup schema. To always access the latest version, use [latest](../squarkup_schema/latest.json).
